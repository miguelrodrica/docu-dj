# Clase 2: Melodía y Contrapunto

Crea melodías memorables y aprende técnicas de superposición.

## Elementos de una buena melodía

- **Rango**: Extensión de notas
- **Contorno**: Forma y movimiento
- **Repetición**: Motivos reconocibles
- **Variación**: Evitar monotonía

## Técnicas de contrapunto

Explora cómo combinar múltiples melodías para crear texturas ricas.

:::tip
Las grandes melodías son simples pero efectivas.
:::
