# Clase 1: Estructura de Canciones

Diseña la estructura de tus canciones para máximo impacto emocional.

## Secciones típicas

- **Intro**: Gancho inicial (8-16 compases)
- **Verso**: Narrativa/contexto (8-32 compases)
- **Pre-coro**: Build-up a lo especial
- **Coro**: Momento memorable (8-32 compases)
- **Puente**: Contraste y variación
- **Outro**: Conclusión

## Equilibrio

Aprenderás a equilibrar estas secciones para mantener el interés del oyente.

:::info
Una buena estructura hace que tus canciones sean más pegadizas y memorables.
:::
