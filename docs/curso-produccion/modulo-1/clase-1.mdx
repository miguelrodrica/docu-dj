# Clase 1: Fundamentos de Música

Aprender los conceptos básicos que necesitas para producir música.

## Elementos fundamentales

- **Notas**: Do, Re, Mi, Fa, Sol, La, Si
- **Escalas**: Mayor, menor, pentatónica
- **Acordes**: Triadas, séptimas, extensiones
- **Ritmo**: Compaz, tempo, subdivisiones

## Importancia para productores

Aunque no necesites saber leer partituras, entender estos conceptos te ayudará a crear melodías memorables.

:::info
La teoría musical te da herramientas para expresar tus ideas de forma más efectiva.
:::
