# Clase 2: Armonía y Progresiones

Crea progresiones de acordes que suenen profesionales.

## Progresiones comunes

- **I-IV-V-I**: Clásica y versátil
- **i-VI-III-VII**: Dramática y cinematográfica
- **II-V-I**: Jazz estándar
- **I-V-vi-IV**: Popular moderna

## Construcción de progresiones

Aprenderás a usar estas progresiones como base para tus tracks y cómo personalizarlas.

:::tip
Las progresiones de acordes son la columna vertebral de tu canción.
:::
