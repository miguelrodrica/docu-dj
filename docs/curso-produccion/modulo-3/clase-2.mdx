# Clase 2: Muestreo y Procesamiento

Utiliza samples existentes y transformalos para crear nuevos sonidos.

## Técnicas de muestreo

- **Grabación de fuentes**: Voces, instrumentos, ambientes
- **Corte y edición**: Extraer porción útil
- **Time stretching**: Cambiar velocidad sin afectar pitch
- **Pitch shifting**: Cambiar tono sin afectar velocidad

## Transformación creativa

Aprenderás a tomar samples comunes y hacerlos únicos.

:::tip
El muestreo es una habilidad que te ahorrar tiempo en producción.
:::
