# Resumen del Módulo 2

Aquí aprenderás composición y estructura para crear canciones más sólidas y memorables.

## Duración estimada

- 60 a 90 minutos de estudio y práctica

## Prerrequisitos

- Haber revisado la introducción del guía
- Tener el software o herramientas base del guía

## Objetivos del módulo

- Comprender los conceptos clave del módulo
- Aplicar lo aprendido en ejercicios prácticos
- Prepararse para avanzar al siguiente módulo

## Qué vas a aprender

- Estructuras de canción
- Desarrollo melódico
- Variación y contrapunto

## Clases del módulo

- Clase 1: Estructura de Canciones
- Clase 2: Melodía y Contrapunto
