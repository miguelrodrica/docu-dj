# Clase 1: Síntesis de Audio

Entende cómo se crean los sonidos sintetizados desde cero.

## Componentes del sintetizador

- **Oscilador**: Genera ondas base (senoidal, triangular, cuadrada, sierra)
- **Filtro**: Modula frecuencias
- **Envolvente**: ADSR (Attack, Decay, Sustain, Release)
- **LFO**: Modulación lenta para movimiento

## Creación de sonidos

Diseñarás sonidos personalizados desde cero usando estos componentes.

:::info
Entender síntesis te da control total sobre tu paleta de sonidos.
:::
