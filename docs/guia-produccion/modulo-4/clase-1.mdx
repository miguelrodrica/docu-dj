# Clase 1: Principios de Mezcla

Aprende a balancear todos los elementos de tu producción.

## Elementos a considerar

- **Volumen**: Balance entre instrumentos
- **Panorámica**: Posicionamiento estéreo
- **Efectos de Reverb/Delay**: Profundidad y espacio
- **Dinámica**: Rango y compresión

## Flujo de trabajo

Seguirás un flujo metódico para mezclar sin perder claridad tímbrica.

:::info
Una buena mezcla hace que tu canción suene profesional en cualquier dispositivo.
:::
