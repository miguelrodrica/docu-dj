# Resumen del Módulo 3

Aquí trabajarás la construcción de sets y la lectura de audiencia para tomar mejores decisiones en vivo.

## Duración estimada

- 60 a 90 minutos de estudio y práctica

## Prerrequisitos

- Haber revisado la introducción del curso
- Tener el software o herramientas base del curso

## Objetivos del módulo

- Comprender los conceptos clave del módulo
- Aplicar lo aprendido en ejercicios prácticos
- Prepararse para avanzar al siguiente módulo

## Qué vas a aprender

- Organización de sesiones
- Curaduría musical
- Adaptación en tiempo real

## Clases del módulo

- Clase 1: Construcción de Sesiones
- Clase 2: Lectura de Audiencia
