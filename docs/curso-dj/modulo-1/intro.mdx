# Resumen del Módulo 1

En este módulo aprenderás los fundamentos del DJing y el equipamiento esencial para empezar.

## Duración estimada

- 60 a 90 minutos de estudio y práctica

## Prerrequisitos

- Haber revisado la introducción del curso
- Tener el software o herramientas base del curso

## Objetivos del módulo

- Comprender los conceptos clave del módulo
- Aplicar lo aprendido en ejercicios prácticos
- Prepararse para avanzar al siguiente módulo

## Qué vas a aprender

- Conceptos base del DJing
- Rol del DJ en eventos
- Equipo necesario para comenzar

## Clases del módulo

- Clase 1: Introducción al DJing
- Clase 2: Equipamiento Básico
