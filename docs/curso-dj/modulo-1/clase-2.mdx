# Clase 2: Equipamiento Básico

En esta clase profundizaremos en el equipo necesario para comenzar a practicar DJing.

## Equipo esencial

- **Tornamesas o CDJ**: Para reproducir música
- **Mixer**: Para mezclar dos fuentes de audio
- **Auriculares**: Para monitoreo
- **Altavoces**: Para la salida de audio principal

## Presupuesto inicial

Aprenderás a elegir equipo según tu presupuesto y nivel de experiencia, desde opciones económicas hasta equipamiento profesional.

:::info
Comienza con lo básico y ve mejorando tu equipamiento a medida que avances.
:::
