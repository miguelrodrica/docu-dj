# Resumen del Módulo 2

Este módulo se centra en técnicas de mezcla para realizar transiciones limpias y mantener la energía del set.

## Duración estimada

- 60 a 90 minutos de estudio y práctica

## Prerrequisitos

- Haber revisado la introducción del curso
- Tener el software o herramientas base del curso

## Objetivos del módulo

- Comprender los conceptos clave del módulo
- Aplicar lo aprendido en ejercicios prácticos
- Prepararse para avanzar al siguiente módulo

## Qué vas a aprender

- Sincronización de tempo (beat matching)
- Control de BPM y pitch
- Transiciones y cambios de pista

## Clases del módulo

- Clase 1: Beat Matching
- Clase 2: Transiciones Básicas
