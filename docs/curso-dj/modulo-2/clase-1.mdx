# Clase 1: Beat Matching

Aprende a sincronizar el tempo de dos canciones para crear mezclas fluidas.

## Conceptos clave

- **BPM**: Beats Per Minute (Pulsaciones por Minuto)
- **Tempo**: Velocidad de la canción
- **Pitch Fader**: Control de velocidad

## Práctica

Practicarás sincronizar dos canciones escuchando atentamente el beat y ajustando el tempo de una canción para que coincida con la otra.

:::info
El beat matching es la base de toda mezcla profesional.
:::
