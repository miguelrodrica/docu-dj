# Resumen del Módulo 4

Este módulo introduce efectos y bases de producción para ampliar tu perfil como DJ.

## Duración estimada

- 60 a 90 minutos de estudio y práctica

## Prerrequisitos

- Haber revisado la introducción del guía
- Tener el software o herramientas base del guía

## Objetivos del módulo

- Comprender los conceptos clave del módulo
- Aplicar lo aprendido en ejercicios prácticos
- Prepararse para avanzar al siguiente módulo

## Qué vas a aprender

- Uso creativo de efectos
- Fundamentos de producción musical
- Flujo de trabajo inicial

## Clases del módulo

- Clase 1: Efectos DJ Básicos
- Clase 2: Introducción a la Producción
