# Clase 2: Transiciones Básicas

Domina las transiciones entre canciones para mantener el flujo musical.

## Tipos de transiciones

- **Corte directo**: Cambio abrupto entre canciones
- **Fade**: Desvanecimiento gradual
- **Crossfade**: Mezcla de dos canciones simultáneamente
- **Loop**: Repetición de una sección

## Ejercicios prácticos

En esta clase ejecutarás transiciones en tiempo real y aprenderás cuándo usar cada tipo según el contexto musical.

:::tip
Las buenas transiciones mantienen el flujo y la energía de la pista.
:::
