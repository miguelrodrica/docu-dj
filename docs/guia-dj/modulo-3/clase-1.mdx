# Clase 1: Construcción de Sesiones

Aprende a organizar y construir una sesión musical efectiva.

## Estructura de una sesión

- **Intro**: Apertura con energía baja
- **Build-up**: Aumento gradual de energía
- **Peak**: Momento de mayor energía
- **Outro**: Cierre de la sesión

## Estrategias de setlist

Descubre cómo seleccionar y ordenar canciones para mantener el interés de la audiencia.

:::info
Una buena programación es el 80% del éxito de una sesión de DJ.
:::
