# Clase 2: Sincronización y Timing

Aprende a sincronizar audio y texto para proyectos multimedia.

## Alineación temporal

- **Sincronización de labios**: Ajustar diálogo a video
- **Timing musical**: Sincronizar efectos con música
- **Marcas de tiempo**: Ubicación precisa de eventos

## Proyecto práctico

Sincronizarás un clip de video con diálogo de audio para crear una escena cohesiva.

:::tip
La sincronización es crítica en la post-producción de video.
:::
