# Clase 2: Exportación y Entrega

Exporta tu proyecto en los formatos correctos según el destino final.

## Formatos de exportación

- **MP3**: Comprimido, ideal para web
- **WAV**: Sin compresión, máxima calidad
- **FLAC**: Comprimido sin pérdida
- **AAC**: Para iTunes y dispositivos Apple

## Configuración de exportación

Aprenderás qué configuraciones usar para cada plataforma de distribución.

:::tip
El formato correcto garantiza que tu audio suene bien donde sea que se reproduzca.
:::
