# Clase 1: Fundamentos de Mezcla

Equilibra múltiples pistas de audio para obtener un sonido cohesivo.

## Parámetros de mezcla

- **Volumen individual**: Nivel de cada pista
- **Panorámica**: Posición estéreo izquierda/derecha
- **Mute/Solo**: Ocultar o aislar pistas
- **Faders**: Control de volumen maestro

## Ejercicio

Mezclarás una canción con voces, guitar, bajo y batería en equilibrio perfecto.

:::info
Una buena mezcla empieza con buenos niveles de volumen.
:::
