# Resumen del Módulo 3

Este módulo aborda mezcla y procesamiento para lograr mayor claridad y calidad sonora.

## Duración estimada

- 60 a 90 minutos de estudio y práctica

## Prerrequisitos

- Haber revisado la introducción del guía
- Tener el software o herramientas base del guía

## Objetivos del módulo

- Comprender los conceptos clave del módulo
- Aplicar lo aprendido en ejercicios prácticos
- Prepararse para avanzar al siguiente módulo

## Qué vas a aprender

- Balance de niveles y paneo
- Uso de EQ y compresión
- Aplicación práctica de efectos

## Clases del módulo

- Clase 1: Fundamentos de Mezcla
- Clase 2: Efectos de Audio
