# Clase 2: Efectos de Audio

Aplica efectos para mejorar y embellecer tus mezclas.

## Efectos comunes

- **EQ**: Ecualización de frecuencias
- **Compressor**: Reduce dinámicas extremas
- **Reverb**: Añade espacio
- **Delay**: Crea repeticiones

## Aplicación práctica

Aprenderás a usar estos efectos sin excederte y manteniendo claridad en tu mezcla.

:::tip
Los efectos deben mejorar, no enmascarar tu audio.
:::
